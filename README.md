## Install

```sh
npm install is-guid-cli -g
```

## Run tests

```sh
npm run test
```

## Usage

```sh
is-guid-cli <string> 
```

```sh
is-guid-cli 8B8AAF97-9AFC-48EC-A04C-F907311F5D44
```
